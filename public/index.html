Zodiacs Portal

Zodiacs Gaming is a full-stack esports community platform built with React, TypeScript, Tailwind CSS, and Supabase (PostgreSQL). It features tournament hosting with real-time brackets (Single Elimination & FFA), guild and team management, player recruitment, quest systems, an in-app shop, and tiered admin moderation — all secured with Row-Level Security policies and powered by Supabase Edge Functions for backend logic. The frontend leverages Vite, TanStack React Query, Framer Motion, and shadcn/ui for a responsive, real-time gaming experience.
